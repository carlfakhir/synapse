# Zettelkasten

Zettelkasten — German for "slip box" — is a personal knowledge management system popularized by sociologist Niklas Luhmann, who attributed his prolific output to a collection of around 90,000 cross-linked index cards. Each note captures one atomic idea and links explicitly to others, building a graph of thought rather than a hierarchy of folders.

The structure resembles how [[associative-memory]] works: no single canonical place for an idea, instead a dense web of connections that makes related concepts easy to surface by following links. Modern tools like Obsidian, Logseq, and Roam Research implement the model digitally.

Unlike [[spaced-repetition]], Zettelkasten doesn't schedule review — it relies on emergent rediscovery through link following.
