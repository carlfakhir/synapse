# Associative Memory

Associative memory is the brain's ability to retrieve information not by a direct address but through patterns of activation that spread across related concepts. Thinking of one idea pulls nearby ideas into mind — nearby in some conceptual space shaped by repeated co-occurrence, emotional weight, and recent use.

Classic psychological models like Collins and Loftus's spreading activation framework describe memory as a graph where activating one node excites its neighbors, with the excitation falling off with distance and age. This is the opposite of static lookup and much closer to how [[embeddings]] and [[semantic-similarity]] work in modern AI systems.

Contrast with [[spaced-repetition]], which targets explicit recall of specific facts, and with [[zettelkasten]], which tries to approximate the brain's web structure with pen and paper.
