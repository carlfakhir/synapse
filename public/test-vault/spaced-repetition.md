# Spaced Repetition

Spaced repetition is a learning technique that schedules reviews at expanding intervals to exploit the spacing effect — the finding that memory is stronger when encoding is distributed over time rather than massed. Systems like Anki, SuperMemo, and Mochi use variants of the SM-2 algorithm to compute each card's next review date from the learner's past performance.

The underlying mechanism is a forgetting curve: memory strength decays exponentially unless reinforced. Optimal review timing targets the moment just before you would have forgotten, turning review into retrieval practice rather than passive re-reading.

Spaced repetition shares theoretical ground with [[associative-memory]] and with note-taking systems like [[zettelkasten]] that emphasize periodic re-encounter.
